**System Requirements**
• Autodesk® Revit® 2022, 2023, 2024, 2025, 2026, or 2027
• Windows 10/11 (64-bit)
• .NET Framework 4.8 (Revit 2022–2024) or .NET 8.0 (Revit 2025–2026) or .NET 10.0 (Revit 2027)
• 100 MB disk space

**Installation**
Run the included `install.bat` — it automatically detects which Revit versions are installed and deploys the add-in to the correct Addins folder. No manual file copying required. Restart Revit after installation.

**Uninstallation**
Run `uninstall.bat` to cleanly remove antiGGGravity from all installed Revit versions.

**Licensing**
antiGGGravity uses hardware-bound license activation. After installation, use the "Hardware ID" button in the AntiGravity ribbon panel to copy your machine identifier and request an activation key from support.

**Design Standards**
The Rebar engine supports international rebar shape standards including AS (Australian Standard), BS (British Standard), and US (ASTM) standard families. Design rules reference NZS 3101 and can be extended for regional codes.
