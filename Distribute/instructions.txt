==================================================
   antiGGGravity Revit Toolkit Implementation
==================================================

How to Install:
--------------
1. Ensure all Autodesk Revit instances are closed.
2. Run "install.bat" by double-clicking it.
3. Type the Revit version year (e.g. 2026) you want to install.
4. Open Revit and check the "antiGGGravity" tab on the ribbon.

How to Uninstall:
-----------------
1. Close Revit.
2. Run "uninstall.bat" by double-clicking it.
3. Type the Revit version year you want to remove.

Support:
--------
For issues or license inquiries, please contact your toolkit administrator.
