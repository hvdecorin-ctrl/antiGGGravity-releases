# antiGGGravity

A powerful Revit C# Add-in specializing in Rebar modeling, view management, and automated visibility toggles.

## Features
- **Rebar Suite**: Advanced automation for Column, Beam, Wall, and Footing rebar.
- **View Filters**: Tools for copying filters between views and generating legend sheets.
- **Visibility Tools**: Rapid toggling of imports, links, and categories.
- **Premium UI**: Modern, glassmorphism-inspired interface for a high-end experience.

## Setup
- Targeted for **Revit 2025/2026** (.NET 8).
- Built with Visual Studio 2022.
